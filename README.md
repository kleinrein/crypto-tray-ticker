# Crypto tray ticker

> A system tray app showing crypto prices written in Go

*Fetches the BTC, LTC, ETH & NEO prices from coincap.io*

## Usage

```go
go get github.com/kleinrein/crypto-tray-ticker
crypto-tray-ticker
```
